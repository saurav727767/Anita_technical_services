ATS Offline Editors Suite
==========================

To run this application offline:

WINDOWS:
1. Extract all files from this ZIP archive into a folder.
2. Double-click the launcher script:
   - "Launch_Office.bat" (for ATS Office Suite)
   - "Launch_Video_Editor.bat" (for ATS Video Editor)
3. The editor workspace will launch instantly in your default web browser.

MOBILE / TABLET (Android & iOS):
1. Extract this ZIP archive on your mobile device.
2. Navigate to the extracted folder and open the HTML file in any browser app (e.g. Chrome, Firefox, Safari):
   - "office.html" (for Word, Sheet, Slides, Paint)
   - "video.html" (for Video editing timeline)
3. Bookmark the page in your browser for easy offline access!
